# Retail Sales Analysis

## Objective
Clean and analyze a café retail transactions dataset to identify key business insights, including total sales performance, comparison of payment methods, and best-selling products, among other relevant metrics.

## Tools
- SQL
- PowerBI

## Dataset
Cafe Sales Dataset: https://www.kaggle.com/datasets/ahmedmohamed2003/cafe-sales-dirty-data-for-cleaning-training

## Project Steps
- Data Cleaning
- Identify Insights
- Dashboard Creation
